# Athus Psi Portfolio 🧠🌲

Uma Landing Page moderna, responsiva e elegante desenvolvida para **Athus Ramalho**, Psicólogo Clínico e Esportivo. O projeto foca em transmitir uma experiência acolhedora (inspirada na natureza) aliada à precisão tática do esporte, refletindo perfeitamente a abordagem do profissional.

## 🚀 Tecnologias Utilizadas

Este projeto foi construído utilizando as melhores e mais modernas ferramentas do ecossistema de front-end:

- **[React](https://react.dev/) + [Vite](https://vitejs.dev/)**: Para uma estrutura base robusta, com componentes reutilizáveis e com um tempo de build/desenvolvimento extremamente rápido.
- **[Tailwind CSS](https://tailwindcss.com/)**: Para estilização ágil e avançada, garantindo um design altamente responsivo, tipografia personalizada e paletas customizadas (Creme, Verde-Escuro, e Destaques Rústicos).
- **[Framer Motion](https://www.framer.com/motion/)**: Utilizado para implementar animações fluídas de entrada, interações com o scroll da página e micro-animações, conferindo ao site um acabamento muito mais premium e engenhoso.
- **[Lucide React](https://lucide.dev/)**: Biblioteca versátil e leve para ícones limpos e minimalistas.

## ✨ Destaques e Funcionalidades

- **Design Responsivo Refinado**: Interfaces cuidadosamente ajustadas para smartphones, tablets (como iPads) e desktops. Adicionamos comportamentos específicos para resoluções intermediárias, mantendo imagens e textos livres de distorções.
- **Identidade Visual Premium**: Diferente da maioria das landing pages simples, o projeto apresenta texturas (efeitos de ruído visual/grain), paletas sutis e fontes mesclando estilo clássico e script.
- **Estrutura de One-Page Moderna**:
  - **Hero Section**: Destaques, brasões flutuantes ajustáveis e botão de CTA.
  - **Sobre Mim**: Detalhes e especializações.
  - **Metodologia / Áreas**: Grade moderna especificando as queixas e o foco.
  - **Atendimento**: Modalidades (Online/Presencial).
  - **Links Úteis & Localização**: Caminhos rápidos de contato, materiais extras e mapa.

## 🛠️ Como Executar o Projeto Localmente

Siga os passos abaixo para baixar, instalar as dependências e experimentar o site na sua máquina:

1. Clone o repositório utilizando o Git:

```bash
git clone https://github.com/JackDev085/athus-psi-portfolio.git
```

2. Navegue até a pasta do projeto clonado:

```bash
cd athus-psi-portfolio
```

3. Instale as dependências via NPM:

```bash
npm install
```

4. Inicie o servidor de desenvolvimento do Vite:

```bash
npm run dev
```

5. Abra seu navegador no endereço: `http://localhost:5173`.
